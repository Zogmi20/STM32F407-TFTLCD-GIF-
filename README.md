# STM32F407 TFTLCD GIF彩色动画显示

基于正点原子探索者STM32F407开发板，在2.8寸TFTLCD上播放28帧动画。

## 📋 项目简介
本项目将一张GIF动图拆分为28帧（每帧72×78像素），转换为RGB565格式的C语言数组，烧录到STM32F407的内部Flash中，在TFTLCD上循环播放。

| 项目 | 参数 |
|------|------|
| 主控芯片 | STM32F407ZGT6 |
| 开发板 | 正点原子 探索者 F407 |
| LCD尺寸 | 2.8寸 TFTLCD (240×320) |
| 动画帧数 | 28帧 |
| 每帧分辨率 | 72 × 78 像素 |
| 颜色格式 | RGB565 (16位色) |
| 总数据大小 | 约307KB |
| 帧率 | 10帧/秒 (100ms/帧) |
